We use a slightly-customized version of Markdown for formatting. Here are the basics:

You type You see
*italics* italics
**bold** bold
[ubnt!](http://ubnt.com) ubnt!
> quoted text
quoted text
* item
* item
* item
  • item
  • item
  • item
# item 1
# item 2
# item 3
  1. item 1
  2. item 2
  3. item 3
Lines surrounded by ``` are treated like code:

```
if 1 * 2 < 3:
    print "hello, world!"
```

Single lines of code can also be written as ` inline code `
~~strikethrough~~ strikethrough
super^script superscript
